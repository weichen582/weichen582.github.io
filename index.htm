Engineering Practicum Intern

Jul 2017 Sep 2017

Implemented a comparison tool for Pagespeed Insights (PSI), an online tool that analyzes web sites and rates their performance. The tool gathers the scores for a large sample URL set rated by two versions of PSI, so as to evaluate the optimized rating system for its rationality, validity and stability.
The tool is designed in a parallel, distributed manner as a Map/Reduce job, which reduces the runtime from dozens of days in a sequential manner to around four hours.
Designed a webpage to provide more friendly user interface, automatically save evaluation results, and query history.
